# lets-learn-go
Learning go, following through examples at go.dev/tour/list

# Setting up the dev environment
Debian as a golang-<version> in its apt repo
 or
Download and untar from https://go.dev/doc/install

Lets use the prepackaged version:

```
$ sudo apt install golang-v1.26
```

Add `/usr/lib/go-1.26/bin` to your path


# Writing code
Initialize the go module
```
$ mkdir hello
$ go mod init example/hello
$ vim hello.go
$ go run .
```
Write some code
